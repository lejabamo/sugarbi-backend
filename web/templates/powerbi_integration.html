Endpoints Disponibles
GET /api/powerbi/schema

Obtener esquema del cubo OLAP

GET /api/powerbi/fact-table

Exportar tabla de hechos

GET /api/powerbi/dimensions

Exportar tablas de dimensiones

GET /api/powerbi/dataset

Crear dataset completo

Ejemplo de Uso
// Obtener esquema del cubo
fetch('/api/powerbi/schema')
  .then(response => response.json())
  .then(data => console.log(data));

// Exportar datos con filtros
const params = new URLSearchParams({
  start_date: '2024-01-01',
  end_date: '2024-12-31',
  zones: 'ZONA1,ZONA2'
});

fetch(`/api/powerbi/fact-table?${params}`)
  .then(response => response.json())
  .then(data => console.log(data));
